## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/expert-connect.git

# Navigate to project directory
cd expert-connect

# Install dependencies
npm install

# Start development server
npm run dev
```
